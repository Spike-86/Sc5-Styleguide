Общая информация по проекту Aservice



Используемые библиотеки:

  1. jQuery 3.3.1 // Манипулирование DOM, работа с AJAX
  2. jQuery UI // Всплывающие подсказки, автозаполнение, datepicker
  3. jQuery cookie // Работа с cookie на стороне клиента
  4. D3.js // Для печати чертежей PDF
  5. DataTables.js // Сортировка таблицы на странице Редактирования скидок клиента
  6. Bootstrap // Управление и анимация DOM элементами
  7. Bootstrap-select.js // Модернизация поведения встроенного select на странице История загрузки прайслистов
  8. Details.polyfill.js // Полифил, добавляющий тег *details* в браузеры, в которых не он не поддерживается
  9. jsPDF.js // Библиотека для генерации PDF файлов на стороне клиента
  10. svg.js // Работа с графикой SVG
  11. SOL.js // Плагин замены стандартных select с функцией поиска и подгрузкой по AJAX
  12. sweet-alert.js // Многофункциональные модальные окна с множеством методов и возможностями дизайна
  13. owl-carousel // Горизонтальный слайдер, используем для отображения банеров
  14. jQuery dateFormat // Преобразование дат
  15. jQuery StorageApi // Библиотека для работы с хранилищем браузера
  16. DateJs // Библиотека высчитывает даты относительно сегодняшнего дня
  17. CreateJs // Библиотека для сторонних слайдеров, получаемых от маркетинга
  18. colResizable.js // Библиотека позволяет динмаически менять ширину столбцов таблицы
  19. select2.js // Библиотека для кастомизации стандартных select
  20. jquery.mb.browser // Библиотека проверяет браузер пользователь браузера и ограничивает доступ
  21. jQuery fancybox // Библиотека для кастомизации всплывающих окон
  22. jQuery Text Editor // Редкатирование HTML с помощью JS
  23. spin.js // Спиннер с анимацией
  24. json-serialization.js // Плагин преобразования JSON в строку
  25. jQuery webui popover // Библиотека для отображения всплывающих окон
  26. jQuery inputmask // Библиотека для валидации вводимых значений в поля input
  27. Form2js // Библиотека для сбора данных в объект JS
  28. date.format.js // Библиотека для преобразования дат
  29. cookieconsent.js // Библиотека для оповещения посетителя сайта об использовании cooikes
